## Tools & Library used

- [Spring Boot 2+]
- [Apache Commons]
- [Spring Data JPA]
- [PostgreSQL Connector]

## Running the application locally

1. Build the project using
   `mvn clean install`
2. Run using
   `mvn spring-boot:run`

## Deploying the application to Tomcat Server

1. Build the project using
   `mvn clean install`
2. Rename the war/jar file
3. Deploy that file to server & restart the server.
